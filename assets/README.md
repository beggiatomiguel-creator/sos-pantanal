## Formatos

### SVG
- Vetoriais para ícones e logos
- Escalável sem perda de qualidade
- Ideal para favicon e ícones pequenos

### PNG
- Com suporte a transparência
- Otimizado para web
- Usado para ícones PWA e screenshots

### JPG
- Compressão eficiente para fotografias
- Usado para Open Graph images

## Otimização

Todos os assets foram otimizados para:
- **Tamanho mínimo**: Compressão sem perda de qualidade perceptível
- **WebP ready**: Prontos para conversão para WebP
- **Responsive**: Tamanhos apropriados para cada uso
